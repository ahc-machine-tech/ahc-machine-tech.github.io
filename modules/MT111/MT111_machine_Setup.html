Introduction ================================== The focus of this lab is to learn how to set up tool lengths and work coordinate system offsets in a CNC mill and lathe in order to be able to machine a part. This lab will cover the following topics: - Jogging all axes of the machine - How to use the tool setter - Setting tool offsets using the tool setter - Using a 3D probe to set X, Y, and Z work offsets. 1 - Tool Length ================================== In this section it will be described how to load the tools into the machine that will be used in our program. How to use different menus in the machine will also be discussed in this section as well as how to jog the different axes of the machine to move the tools to a desired location.
